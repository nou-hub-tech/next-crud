# Next.js CRUD Topic App

## Overview

This project is a CRUD application for managing topics with a Next.js frontend and a MongoDB-backed API.

## What It Covers

- App Router structure
- Create, read, update, and delete flows
- REST API routes for topics
- Form handling for adding and editing topics
- Responsive interface built with Tailwind CSS
- Database integration with MongoDB

## Routes

- /
- /api/topics
- /api/topics?id=ID
- /api/topics/ID
- /add-topic
- /edit-topic/ID

## User Capabilities

- View a list of topics
- Add a new topic
- Edit an existing topic
- Delete a topic

## Development

Run the development server with:

```bash
npm run dev
```

Open the local application in your browser at http://localhost:3000.
